# Program 1: LEX Program to Identify Tokens

## Aim

To identify keywords, identifiers, numbers, and operators using LEX.

## Description

This program uses LEX to scan input and classify tokens into:

* Keywords
* Identifiers
* Numbers
* Operators

## Keywords Used

int, float, char, double, if, else, while, for, return

## Sample Input

int a = 10 + 20;

## Sample Output

Keyword: int
Identifier: a
Operator: =
Number: 10
Operator: +
Number: 20

## Tools Used

* Flex (LEX)
* GCC
