# Program 3: Count Lines, Words and Characters

## Aim

To count the number of lines, words, and characters using LEX.

## Description

This program uses LEX to scan the input text and count:

* Lines
* Words
* Characters

## Logic Used

* Lines are counted using newline (`\n`)
* Words are identified as sequences of characters separated by spaces or tabs
* Characters include all input symbols including spaces and newline

## Sample Input

Hello world
This is ATC lab

## Sample Output

Lines: 2
Words: 5
Characters: 28

## Tools Used

* Flex (LEX)
* GCC
