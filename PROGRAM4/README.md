# Program 4: Validate Regular Expression

## Aim

To check whether a given input is a valid regular expression using LEX.

## Description

This program validates a regular expression by checking:

* Allowed characters (alphabets, digits)
* Operators: +, *, |
* Balanced parentheses

If any invalid symbol is found or parentheses are unbalanced, the expression is considered invalid.

## Sample Input

(a+b)*c

## Sample Output

Valid Regular Expression

## Tools Used

* Flex (LEX)
* GCC
