# Program 5: Arithmetic Expression Evaluation using YACC and LEX

## Aim

To evaluate arithmetic expressions using YACC with LEX as tokenizer.

## Description

This program uses:

* LEX to identify numbers and operators
* YACC to parse and evaluate arithmetic expressions

It supports:

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Parentheses

## Sample Input

3+5*2

## Sample Output

Result = 13

## Tools Used

* Flex (LEX)
* YACC
* GCC
