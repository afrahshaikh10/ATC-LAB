# Program 7: Abstract Syntax Tree using YACC

## Aim

To construct an Abstract Syntax Tree (AST) for arithmetic expressions using YACC.

## Description

This program builds an AST where:

* Each internal node represents an operator
* Each leaf node represents an operand

The tree is printed using inorder traversal.

## Sample Input

3+5*2

## Sample Output

Inorder Traversal of AST:
3 + 5 * 2

## Tools Used

* Flex (LEX)
* YACC
* GCC
