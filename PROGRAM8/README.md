# Program 8: Control Statements Parser using LEX and YACC

## Aim

To parse control statements like if, if-else, and while using LEX and YACC.

## Description

This program uses:

* LEX for tokenizing keywords, identifiers, and operators
* YACC for parsing control statement grammar

It validates:

* IF statements
* IF-ELSE statements
* WHILE loops

## Sample Input

if(a<b) x; else y;

## Sample Output

Valid IF-ELSE statement

## Tools Used

* Flex (LEX)
* YACC
* GCC
